# Guide d'utilisation

Bienvenue dans le manuel d'utilisation complet de la plateforme.

---

## Compatibilité

La plateforme est accessible sur **ordinateur et téléphone**, l'interface s'adapte automatiquement à la taille de l'écran.

Néanmoins, pour les fonctionnalités d'administration (création et modification de challenges), il est **recommandé d'utiliser un ordinateur** pour une meilleure expérience.

> Ce guide utilise la version ordinateur pour les captures d'écran, mais le fonctionnement reste identique sur téléphone.

---

## Page d'accueil

![Accueil](./../images/preview/desktop.png)

La page d'accueil est la première page visible lorsque vous n'êtes pas connecté.

### Header

Le header est **toujours visible** quelle que soit la page consultée. Il est composé de :

* **FrameLab** (à gauche) — logo cliquable qui ramène à l'accueil
* **Challenges**, **Hall of Frames**, **Boutique** (à droite) — ancres qui font défiler la page vers les sections correspondantes
* **Se connecter** — bouton principal pour accéder à son compte

### Footer

Le footer est également **toujours visible** en bas de chaque page. Il est divisé en quatre parties :

* **Navigation** — reprend les mêmes options que le header (Challenges, Hall of Frames, Boutique)
* **Légal** — Mentions légales, CGU, Politique de confidentialité
* **Contact** — formulaire pour envoyer un message

> L'adresse de contact est fictive, merci de ne pas envoyer de message.

## Créer un compte

![Inscription](./../images/preview/signup.png)

Pour accéder aux fonctionnalités de la plateforme, vous devez créer un compte. Renseignez les informations suivantes :

* Nom
* Prénom
* Adresse e-mail
* Mot de passe
* Confirmation du mot de passe

> **Important — Pour votre sécurité, il est fortement recommandé de NE PAS utiliser vos véritables informations personnelles lors de l'inscription.** Ce site est réalisé dans le cadre de l'épreuve E6 et n'a pas vocation à être maintenu dans le temps.

Une fois les champs remplis, cliquez sur **Créer mon compte**.

Si le mot de passe et sa confirmation concordent, vous serez redirigé vers une page de validation.

### Validation du compte

![Validation](./../images/preview/validation.png)

Le site n'envoie pas d'e-mail de confirmation. La validation du compte s'effectue directement depuis cette page en cliquant sur le bouton **Valider mon compte**.

### Connexion

![Connexion](./../images/preview/login.png)

Une fois le compte validé, vous êtes automatiquement redirigé vers la page de connexion. Renseignez votre adresse e-mail et votre mot de passe, puis cliquez sur **Se connecter**.

## Accueil connecté

![Accueil connecté](./../images/preview/desktop_logged.png)

Une fois connecté, le header évolue :

* **Challenges** redirige désormais vers la page des challenges
* **Hall of Frames** redirige vers la page du classement
* **Mon espace** apparaît à côté de Boutique et redirige vers votre profil
* **Se connecter** devient **Se déconnecter**

## Page Challenges

![Challenges](./../images/preview/challenge.png)

Cette page présente le challenge en cours ainsi que les challenges passés.

### Challenge en cours

La carte du challenge en cours affiche :

* Le temps restant avant la fin du challenge
* Le nom et la description du challenge
* Le bouton **Participer maintenant** — redirige vers la page de téléchargement de l'application de bureau, car la participation à un challenge s'effectue uniquement via l'application de bureau
* Le bouton **Voir les soumissions** — redirige vers les participations du challenge

### Archives

Une carte dédiée permet d'accéder aux challenges passés et à leurs participations.

### Panel d'administration

Si votre compte est administrateur, un bouton **Accéder au panel** est visible sur la page. Il permet de gérer les challenges.

---

## Participations d'un challenge

![Participations](./../images/preview/challenge_participations.png)

Si un challenge a des participations, chacune est affichée sous forme de carte dépliable.

### Carte repliée

Lorsqu'une carte est repliée, elle affiche :

* Le nom de l'auteur
* La date de soumission
* Le score (uniquement si vous avez déjà voté pour cette participation)

### Carte dépliée

Lorsqu'une carte est dépliée, elle affiche :

* Les commentaires laissés sur la participation
* Les administrateurs peuvent supprimer les commentaires

### Voter

Si vous n'avez pas encore participé au challenge, vous pouvez voter pour les participations des autres membres. Il suffit d'attribuer un nombre de points pour chacun des trois critères suivants :

* Créativité
* Technique
* Respect du thème

Puis de cliquer sur **Voter** pour enregistrer votre vote.

## Panel d'administration — Challenges

![Administration](./../images/preview/challenge_admin.png)

Ce menu est réservé aux administrateurs.

### Modifier un challenge

Sélectionnez un challenge dans la liste pour modifier ses informations :

* Titre
* Description
* Date de début et de fin
* Image

Cliquez sur **Valider** pour enregistrer les modifications.

Il est également possible d'archiver ou de désarchiver un challenge depuis ce menu.

> Si un challenge est archivé pour la première fois, il est considéré comme terminé : le classement final est établi et les points de récompense sont attribués aux participants.

### Créer un challenge

Cliquez sur **Nouveau challenge**, renseignez les informations puis cliquez sur **Valider** pour créer le challenge.

Pour programmer un challenge dans le futur, il suffit de renseigner une date de début ultérieure.

### Supprimer un challenge

Cochez la case située avant l'identifiant du challenge à supprimer, puis cliquez sur **Supprimer**.

### Recherche et tri

Il est possible de rechercher un challenge par titre ou description, et de trier la liste par identifiant par ordre croissant ou décroissant.

## Mon espace — Profil

![Profil](./../images/preview/profil.png)

La page de profil permet de gérer vos informations personnelles :

* Nom et prénom
* Adresse e-mail
* Mot de passe

Elle affiche également un historique de vos participations et de vos votes.

Il est aussi possible de consulter le profil des autres membres depuis cette page.

### Confidentialité des scores

* Tant qu'un challenge n'est pas terminé, si vous avez participé, votre score n'est pas visible, même depuis votre propre profil.
* Si vous avez voté pour un challenge en cours, les scores que vous avez attribués sont visibles uniquement sur votre propre profil — ils n'apparaissent pas pour les autres membres.
* Une fois le challenge terminé, tous les scores deviennent visibles pour l'ensemble des membres.

## Hall of Frames

![Hall of Frames](./../images/preview/hallofframe.png)

La page Hall of Frames présente le classement des **10 meilleurs contributeurs** de la plateforme.

Les trois premiers occupent une place à part :

* **1re place** — profil affiché en or
* **2e place** — profil affiché en argent
* **3e place** — profil affiché en bronze
